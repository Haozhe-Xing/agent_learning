# 第二部分：核心能力篇

> 🧠 系统掌握构建 Agent 所需的全部核心技术能力，从工具调用到强化学习训练。

本部分是全书最厚实的一部分，包含 7 章，覆盖了 Agent 的完整能力进阶链。

| 章节 | 内容 | 类比 |
|------|------|------|
| **第3章** 工具调用 | Function Calling、自定义工具 | Agent 的「手」 |
| **第4章** 记忆系统 | 短期/长期/工作记忆 | Agent 的「脑」 |
| **第5章** 规划与推理 | ReAct、任务分解、反思 | Agent 的「思维」 |
| **第6章** RAG | 文档检索、向量数据库 | Agent 的「知识库」 |
| **第7章** 上下文工程 | 窗口管理、注意力预算 | Agent 的「信息中枢」 |
| **第8章** Harness 工程 | Agents.md、六大工程支柱 | Agent 的「规范手册」 |
| **第9章** 技能系统 | 技能封装、学习与发现 | Agent 的「技能树」 |

**能力进阶路线**：会用工具 → 会记忆 → 会思考 → 会检索 → 会管理上下文 → 会遵循规范 → 会封装技能
